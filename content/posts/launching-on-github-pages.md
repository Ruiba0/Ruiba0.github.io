## 背景

  自从年后开始就一直在高强度 vibe coding，一直想搞个小demo玩一玩，上个月突然想到正好可以搞一个博客，于是便有了这个博客网站。没有集中时间一下子搞完，用零碎时间想起来了就跑一会儿。

  这算是我的第二个博客网站了，第一个则是大一时刚入学没多久，对着b站和网上的教学一步步用 Hexo + Github Pages 搭建第一个博客，还记得上机课时，能通过机房的电脑访问我的第一个小网站，是非常兴奋的。虽然后面这个荒废掉了...

## 技术实现

  主要通过codex + gpt-5.4 来完成，尝试用了很多前端开发效果的skill，一开始搞的花里胡哨的，后来删删减减，并且搭配我个人开发习惯的skill完成。技术选型是 Next.js 16 + React 19 + TypeScript，部署方式同样选用Github Pages 部署，全程 vibe coding，从0开始。

## 结语

  后面会慢慢分享一些东西在这个博客上，也算是写给自己当一个小小的记事本了。
